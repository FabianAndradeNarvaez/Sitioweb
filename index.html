Descripción del OVI

Cordial saludo estimado (a) estudiante, sea bienvenido al OVI (Objeto Virtual de Información), sobre los temas de Git y GitHub. En el cual encontrara material de lectura y contenidos audiovisuales que le ayudaran en el desarrollo de las actividades del curso Diseño de sitios Web de una manera práctica y didáctica.

La información de este OVI es fundamental para el aprendizaje sobre el manejo de Git y GitHub, ya que contiene material que le ayudara a comprender los conceptos básicos y funcionalidades. Además le será de gran ayuda en el transcurso del curso Diseño de sitios Web.

En este OVI en encontrará información sobre:

• Uso de Git y GitHub.

• Instalación de Git y GitHub.

• Crear Repositorios en GitHub.

Objetivos del OVI

Cordial saludo estimado (a) estudiante, sea bienvenido al OVI (Objeto Virtual de Información), sobre los temas de Git y GitHub. En el cual encontrara material de lectura y contenidos audiovisuales que le ayudaran en el desarrollo de las actividades del curso Diseño de sitios Web de una manera práctica y didáctica.

• Dar a conocer al estudiante el uso de Git y GitHub.

• Aprender a instalar Git y GitHub en Windows.

• Crear y configurar repositorios en GitHub.
